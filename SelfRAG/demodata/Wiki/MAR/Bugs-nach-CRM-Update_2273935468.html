Standardansicht anpassen

Gibt es jedoch auch eine Möglichkeit die kompakte Ansicht anzupassen und die Reihenfolge der verschiedenen Module zu ändern? (Dass man einzelne Felder an- oder abwählen kann für die kompakte Ansicht, habe ich gesehen.)

[crm-now] leider verstehe ich nicht ganz was Sie meinen. In der kompakten Detailansicht können Sie, wie Sie ja schon bemerkten, Felder hinzufügen bzw. entfernen. Module gibt es da nicht. Aber die kompakte Detailansicht hat, wie auch die Detailansicht auf der rechten Seite s.g. bezogene Listen, die bezogene Informationen aus anderen Modulen zur Ansicht bringen. Meinen Sie die?  Die angezeigten Module entsprechen den freigeschalteten Modulen aus dem Module Manager und werden angezeigt, wenn es eine Referenz geben kann. Die bezogenen Listen können Sie selbst einstellen (Einstellungen -> Felder bearbeiten -    bezogenen Listeninhalt arrangieren )


Die Nutzer hätten es am liebsten, dass wie beim alten Vtiger alle bezogenen Module bei den Organisationen und bei den Personen komplett angezeigt werden auf der kompakten Ansicht, ohne dass man rechts drauf klicken muss.

[crm-now] das habe ich leider nicht verstanden

→ Kompakte / Zusammenfassungs-Ansicht: Widgets-Position angeben, wird programmiert


[crm-now] Jeder Benutzer kann im Menü „Meine Einstellungen“ über das Feld „Standardansicht“ selbst festlegen, welche Ansicht zuerst gezeigt werden soll. So eine Einstellung gilt aber dann für alle Module (es gibt aber nicht bei allen Modulen eine kompakte Ansicht). Reicht das für Ihre Zwecke aus?

Wenn Sie jedoch möchten, dass die Detailansicht immer bei Organisationen und Personen fest eingestellt werden soll (unabhängig davon, was ein Nutzer individuell einstellt), so müssten wir umprogrammieren. Das wäre ein Aufwand von ca. 1h (99 EUR);
