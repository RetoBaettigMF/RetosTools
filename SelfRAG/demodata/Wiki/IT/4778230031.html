IT : Verhindern der Windows 10 Update Notification in Windows 7 & 8.1

Created by Sandro Brandt (Unlicensed), last modified by Hans Zemp on Jan. 04, 2024

Um zu verhindern, dass die Windows Update Notification wiederholt angezeigt wird, kann ein Windows Update deinstalliert werden. Es gibt mehrere Massnahmen um die Notification zu verhindern. Diese ist am Einfachsten (es müssen keine zusätzlichen Tools installiert werden) und es hat für mich (BRS) funktioniert.

Entfernen des Updates KB3035583

Um das Update zu entfernen muss Windows Update in der Systemsteuerung geöffnet werden. In diesem Fenster muss unten links der Link Installierte Updates angeklickt werden:

Am einfachsten findet man das entsprechende Update wenn man die Liste nach Namen sortiert und nach Update for Microsoft Windows (KB3035585) sucht.

Mit einem Rechtsklick auf das Update und der entsprechenden Bestätigung kann dieses deinstalliert werden. Danach muss ein Neustart durchgeführt werden.

 Update deaktivieren

Um zu verhindern, dass das Update automatisch wieder installiert wird muss nach verfügbaren Updates gesucht werden. Wenn das Update KB3035583 erneut angezeigt wird, kann dieses mit einem Rechtsklick darauf und der Auswahl Update ausblenden inaktiviert werden. 

Automatisches Windows 10 Update

Vermehrt haben User davon berichtet, dass sich Windows 10 automatisch auf ihren Rechnern installiert hat. Nach Meldung von Microsoft hat es sich dabei um einen Fehler gehandelt, dass das optionale Windows 10 Update per Default angekreuzt war. Dies wurde von Microsoft behoben. Es sollte deshalb nicht mehr vorkommen, dass sich das Windows 10 Update selbstständig installiert.
