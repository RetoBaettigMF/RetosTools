IT : Windows GodMode aktivieren

Created by Ehemaliger Benutzer, last modified on Aug. 05, 2015

Mit dem versteckten GodMode können unter Windows sämtliche Einstellungsmöglichkeiten auf einen Blick - aufgeteilt in Kategorien -  angesehen und angepasst werden. 

Vorgehen zur Aktivierung:

  1. Ordner erstellen
  2. Ordner umbenennen in:  GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}
  3. Der String vor dem Punkt ergibt nachher den Namen des Aufrufes, muss nicht unbedingt so pathetisch sein.

 

Der GodMode funktioniert unter Windows 7, 8 und 10.
