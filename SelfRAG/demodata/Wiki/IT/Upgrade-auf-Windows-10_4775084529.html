IT : Upgrade auf Windows 10

Created by Ehemaliger Benutzer, last modified by nizami.sevindi on Aug. 26, 2024

Möglichkeiten

  • Unsere Windows 7 oder Windows 8/8.1 Lizenzen können gratis auf Windows 10 updated werden. (Hinweis: ab 29. Juli 2015 bietet Microsoft das Windows 10 Upgrade für 1 Jahr gratis an!)
  • Ein Upgrade geschieht über eine laufende Windows Installation und funktioniert in der Regel problemlos - d.h. nach dem Upgrade kann direkt weitergearbeitet werden.
  • Wer von Windows 7 auf Windows 10 umsteigen will und ein Clean Install (d.h. Windows komplett frisch aufsetzen) plant, der muss trotzdem zuerst ein Upgrade des laufenden Windows 7 machen. Dabei wird die Windows 7 Seriennummer auf dem Aktivierungsserver aktiviert und mit der Hardware verknüpft. Erst danach ist ein Clean Install möglich.

Vorgehen

Variante A (für Windows 8/8.1 Benutzer)

  • Windows 10 über Windows Taskleisten-Symbol reservieren -> sobald es verfügbar wird (kann Stunden/Tage dauern da Microsoft stufenweise Downloads freischaltet), kann das Upgrade eingespielt werden.

Variante B (für Windows 7/8/8.1 Bennutzer, schneller)

  • ISO Image auf Netzlaufwerk nehmen und mit z.B. Rufus einen bootable USB-Stick (mind. 8GB) erzeugen.
    G:\Geteilte Ablagen\SW-Ressources_and_Virtual_Images\SW-Ressourcen\3_Engineering\Windows 10\Windows10Pro_x32_x64.iso
  • Alternativ: ISO mit ISO-Tool (z.B. PowerISO) mounten und Upgrade installieren.

Wichtige Hinweise

  • Wer virtuelle Maschinen basierend auf Oracle VirtualBox benutzt soll mit dem Upgrade noch zuwarten -> VMs können mit der aktuell verfügbaren Version (5.0.1) unter Windows 10 nicht gestartet werden!
