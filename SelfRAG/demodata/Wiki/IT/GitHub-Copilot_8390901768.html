IT : GitHub Copilot

Created by Reto Bättig, last modified by Hans Zemp on Sept. 18, 2024

Wir stellen für Entwickler bei Bedarf GitHub Copilot Lizenzen zur Verfügung.

Die Lizenz kostet ca. $20/Monat und kann bei Bedarf einfach per Email an it@m-f.ch bestellt werden.

Bitte erstellt einen Account unter:

https://github.com/

mit einem Name wie “HansZempCudos” und meldet Euch dann bei der IT, wo Euch dann eine Lizenz zugeteilt wird.

Wenn ihr den Copilot nicht mehr verwendet, meldet Euch auch bei der IT, damit wir das Abo stornieren können.

Die Installation von Copilot ist unproblematisch für VisualStudio und VisualStudio Code und hier beschrieben: https://docs.github.com/en/copilot/configuring-github-copilot/configuring-github-copilot-in-your-environment?tool=visualstudio

Sicherheit/Geheimhaltung

Die Verwendung von GitHub Copilot ist gemäss der GitHub PrivacyPolicy relativ unbedenklich: https://docs.github.com/de/site-policy/privacy-policies/github-copilot-for-business-privacy-statement

Es werden zwar Code-Schnipsel an Copilot gesendet für die Berechnung der Vorschläge, diese werden jedoch nicht gespeichert oder anderweitig weiterverwendet:
