Name im Netzwerk: CudosNAS

Adresse für Windows Explorer: \\192.168.1.10\public

username: public

Sicherheitshinweis

Vom Public Verzeichnis wird kein Backup erstellt und es ist sichtbar für jede Person, welche Zugriff auf das Cudos Netzwerk hat (nicht über WLAN).

Es ist als Ersatz für das ehemalige Global Temp Verzeichnis T: auf \\mfwinsrv01\GlobalTemp gedacht

Das Laufwerk kann mit folgendem Befehl bei Bedarf wieder als T: verbunden werden:

net use T: \\192.168.1.10\public

NAS Infos

WD My Cloud Mirror Gen2 (2 x 3TB, WD Red) - Kosten: Fr. 289 von Digitec

Adresse für die Administration: http://192.168.1.10

Die Adresse wurde im DHCP-Server fix zugeteilt

Admin login: admin

Admin passwort: Siehe "G:\Geteilte Ablagen\Diverses\Administration\IT\IT Passwörter.docx"

Das NAS ist im Raid 1 Modus konfiguriert (Mirroring) und sendet im Problemfall eine email an it@m-f.ch
