IT : Google Drive nicht mehr vorhanden

Created by Hans Zemp, last modified on Dez. 11, 2020
  • Über den Task Manager alle "Google Drive File Stream" Prozesse löschen, damit der User Folder frei wird
  • %USERPROFILE%\AppData\Local\Google\DriveFS (z.B. C:\Users\zeh\AppData\Local\Google\DriveFS) löschen
  • Google Drive File Stream neu installieren (siehe "Google Drive File Stream Installation und Verzeichnissstruktur").
