Admin Zugriff auf ALLE Azure Ressourcen

Problem: Ein neuer Admin braucht Zugriff auf die Azure Subscriptions.

Lösung: https://learn.microsoft.com/en-us/azure/role-based-access-control/elevate-access-global-admin

Microsoft Entra ID → Properties → Access management for Azure resources → Yes

Zugriff eines Mitarbeitenden auf eine spezifische Azure Ressource

Beispiel: Zhuo Zugriff geben auf die mf-openai Subscription

  • mf-openai Subscription öffnen

  • Access Contrtrol (IAM) anklicken links

  • Role Assignments => Add (oben, links der Mitte)

  • Unter “Members” den MA auswählen und hinzufügen

  • Unter Role auf “Privileged administrator roles” klicken und dann die Rolle wählen (“Contributor” reicht meist, sonst “Owner”
