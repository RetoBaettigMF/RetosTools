IT : Evaluation Dateispeicher (inStep Ersatz)

Created by Reto Bättig, last modified by nizami.sevindi on Aug. 26, 2024

Ausgangslage

Alle Projektdaten werden heute in inStep gespeichert. Hauptfeatures sind:

  • Zentraler Zugriffsort auf alle Projektdaten (von überall per VPN erreichbar)
  • Versionierung der Dateien (wiederherstellen alter Versionen möglich)
  • Zugriffskontrolle (es können nicht 2 Leute gleichzeitig an einem Dokument arbeiten)

Hauptnachteile sind:

  • Nur per VPN erreichbar
  • Relativ umständlich, um eine Datei rein zu bringen oder anzupassen (aufwändiger, als einfach auf einem Share wie z.B. P:\ eine Datei zu bearbeiten)
    • Programm öffnen
    • File auschecken
    • File bearbeiten
    • File einchecken
