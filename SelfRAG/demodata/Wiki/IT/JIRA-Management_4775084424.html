IT : JIRA Management

Created by Ehemaliger Benutzer, last modified by nizami.sevindi on Aug. 26, 2024

JIRA Cloud Instanz: mfsupport.atlassian.net

Zweck

Diese JIRA Instanz wurde primär eingerichtet als Supportportal für Kundenprojekte.

Aktuell werden drei Projekte für den Geberit Servicevertrag aktiv genutzt:

  • Geberit AC Mera Support (ACMERA)
  • Geberit AquaClean EOL Support (AQUACLEAN)
  • Geberit allgemeiner Support (GAS)
  • Geberit Qualitätsanalysesystem Support

Zugriffsrechte

Sofern für interne Projekte JIRA verwendet wird, muss in den Projekt Settings das Permission Scheme manuell auf Default Permission Scheme
gesetzt werden. Bei neu erstellten SW-Projekten wird das Default Software Scheme gewält, was die Einsicht für alle User erlaubt - d.h. auch für
allfällige Kunden mit JIRA-Zugang! 

Interne JIRA Projektbeispiele:

  • ABB-Wettingen
  • MF-8098 Bystronic SSC
