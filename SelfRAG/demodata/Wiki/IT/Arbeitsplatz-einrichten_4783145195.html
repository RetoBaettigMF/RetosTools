IT : Arbeitsplatz einrichten

Created by Reto Bättig, last modified by nizami.sevindi on Aug. 26, 2024

Die Standardkonfiguration, sowie die Software die auf dem Arbeitsrechner installiert werden muss ist unter Beschaffung und Einrichtung Infrastruktur für neue MA ersichtlich.

Rechnername

Kurzzeichen-PC (z.B. ZEH-PC)

Der neue Mitarbeiter sollte mit Angaben zu Windows und Office Lizenzen in folgendem Google Spreadsheet eingetragen werden:

https://docs.google.com/spreadsheets/d/1mpk4RUN3OW43q-hWzwfV7mc0LQhmZ2IVLMeefLl0AmQ/edit?usp=sharing

Betriebssystem

  • Auf neuen Rechnern wird Windows 10/11 Pro installiert. Es es vor allem beim Kauf darauf zu achten, dass zwingend die Pro Version enthalten ist,
    da mit Windows Home Features wie Bitlocker, Remote Desktop und HyperV fehlen (Featurevergleich, siehe hier).

  • Bestehende Mitarbeiterrechner welche mit Windows 7 oder 8.1 laufen, können von den Mitarbeitern selbständig updated werden, das Update wird
    jedoch nicht verlangt. (Der Umstieg auf Windows 10/11 Pro soll vorrangig bei der Beschaffung eines neuen Rechners erfolgen.)

  • Ein Windows 10 Pro ISO-Image (Anniversary Update, Version 1607) ist hier abgelegt:
    G:\Geteilte Ablagen\SW-Ressources_and_Virtual_Images\SW-Ressourcen\3_Engineering\Windows 10

  • Ein Windows 11 Pro ISO-Image (22H2) ist hier abgelegt:
    G:\Geteilte AblageSW-Ressources_and_Virtual_Images\SW-Ressourcen\3_Engineering\Windows 11

  • Ein mit Rufus geschriebener bootbarer Win10 Pro Image USB-Stick ist im IT-Lizenzschrank bei den Windows-Lizenzen abgelegt. (bitte nach Gebrauch jeweils wieder retournieren).

Lokaler Administrator

Der Arbeitsrechner soll standardmässig mit folgendem lokalen Administrator eingerichtet werden:

Benutzername

admin

Passwort

 Siehe G:\Geteilte Ablagen\Diverses\Administration\IT\IT Passwörter.docx, lokaler Admin-Zugang

Nachdem ein Login über einen MS-Account erfolgte, muss der Rechnername verwendet werden, um sich als “admin” anmelden zu können:

  • z.B. ZEH-PC\admin
