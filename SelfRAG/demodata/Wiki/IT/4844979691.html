IT : 3CX Telefonzentrale Bedienung für Mitarbeitende

Created by Reto Bättig, last modified by Ruedi Steinmann on Nov. 10, 2023

Telefonieren vom PC aus

Webclient

Die Benutzer kriegen von der IT eine Email, in welchem die Zugangsdaten (Extension Number, Pin, Passwort) angegeben sind.

Ebenso ist der Link zum Webclient angegeben: https://mfengineering.3cx.ch:5001/webclient

Der Webclient funktioniert am besten mit Chrome. 

Einstellung der Audio-Geräte und Kamera

Lautsprecher, Mikrofon und Kamera können eingestellt werden im Webclient unter

Einstellungen→ Audio/Viceo

Paralleles Weiterleiten an Mobilgerät:

Anrufe können parallel an das Mobiltelefon weitergeleitet werden. So kann man auch von unterwegs oder im Homeoffice Anrufe auf die Geschäftsnummer entgegen nehmen:

  • Webclient → Einstellungen → Allgemein: Mobilnummer eintragen
    Wichtig: Die Handynummer nur als Zahl, ohne Spaces eintragen!

  • Webclient → Einstellungen → Status

Spezialfunktionen

Weiterleiten eines Anrufes

Wenn man einen Anruf angenommen hat, erscheint ein Fenster, mit welchem man Anrufe weiterleiten kann:

Hier kann man die Anrufe 

  • Vermitteln (unattended): Der Anruf wird direkt an eine Nummer weitergeleitet und wenn die Person abhebt, hat ist sie direkt mit dem weitergeleiteten Anrufer verbunden.

  • Makeln (attended): Man ruft die andere Nummer an. Wenn jemand abnimmt kann man sagen, wer am anderen Ende der Leitung ist und dann aufhängen => der Anruf wird beim Aufhängen verbunden

  • Konferenz: Man kann zusätzliche Anrufe hinzunehmen

Annehmen eines Gespräches von einer fremden Nummer:

Wenn irgend ein Telefon klingelt, sieht man das im Switchboard, wenn beim User ein gelber oder roter Punkt erscheint. Man sieht auch, wer der Anrufer ist. Wenn man auf die Nummer des Anrufers klickt, hat man die Option, den Anruf zu sich zu holen. Wenn man auf Annehmen klickt, klingelt das eigene Telefon, man kann abheben und hat die Verbindung.

Chrome-Erweiterung

Für Chrome existiert auch eine sehr gute Erweiterung. Wir empfehlen, diese zu installieren. Dazu oben links unten "Apps" klicken, Erweiterung installieren, Erweiterung aktivieren ("Hinzufügen"):

=> 


Mit der Erweiterung können Telefonnummern im Broswer direkt angeklickt werden für ein Telefonat.

Man ist auch erreichbar, sobald man den Browser geöffnet hat und man muss den Webclient gar nicht öffnen.

Die Audio-Einstellungen sind zu finden unter dem "Hamburgermenu" => Einstellungen => Personalisieren

Telefonbuch

Das Telefonbuch wird in regelmässigen Abständen von vTiger importiert. Wichtig: Keine eigenen Adressen im Telefonbuch der 3CX-Anlage speichern, da diese beim Import vom vTiger gelöscht werden. Wenn schon müssen die neuen Nummern im vTiger erfasst werden. Bei Bedarf für eine Auffrischung des Telefonbuches einfach die IT anfragen.

Fernsteuern von Geräten

Das Tischtelefon, Handy etc. kann "ferngesteuert" werden, wenn im Menu oben Rechts das entsprechende Endgerät ausgewählt wird:

  • Snom320: Wenn man vom Webclient aus anruft, wird die Verbindung über das Tischgerät hergestellt

  • Browser-Erweiterung (Kommt nur, wenn man sie installiert hat - siehe weiter unten)

  • Smartphone; Die Verbindung wird über die 3cx App auf dem Mobilgerät hergestellt und man kann damit telefonieren

Mobile APP

Um von unterwegs über die M&F Nummer telefonieren zu können, kann die Mobile APP 3cx (Android: https://play.google.com/store/apps/details?id=com.tcx.sipphone14, IOS:https://apps.apple.com/us/app/3cx/id992045982) installiert werden.

Die Links sowie der zur Installation nötige Matrix-Code ist ebenfalls im Willkommens-Email zu finden. Nachdem der Matrix-Code gescannt ist, sollte bereits alles richtig eingerichtet sein.

Wir empfehlen die Verwendung der mobilen App nicht, weil sie nicht gut funktioniert. Insbesondere werden immer aus versehen irgendwelche Buttons gedrückt, wenn man das Telefon ans Ohr nimmt!

Besser ist es, Die Anrufe parallel an die Handynummer weiterzuleiten (siehe Kapitel "Paralleles Weiterleiten an Mobilgerät:" weiter oben.
