IT : Staging Webseite für grössere Änderungen

Created by Anna Laura John on Nov. 17, 2020

Für grössere Änderungen an unserer Website, wie zum Beispiel das Aktualisieren der PHP Version, sollten die Änderungen zuerst auf einer Test-Website getestet werden. So kann man sichergehen, dass das Verfügbarkeit der Website nicht unterbrochen wird.

Um die Test-Website aufzusetzen, wird die Domain testing.m-f.ch gebraucht. Die Testing Domain ist normalerweise eine Weiterleitung auf www.m-f.ch. Um sie als Test-Webseite zu gebrauchen, sollten folgende Schritte auf Metanet ausgeführt werden.

Staging Webseite Aufsetzen:

  1. Weiterleitung von testing.m-f.ch auf www.m-f.ch aufheben
  2. Die aktuelle Webseite auf testing.m-f.ch kopieren
  3. Die Datenbank mf_website in die Datenbank testing_mf_website kopieren
  4. testing.m-f.ch die Datenbank testing_mf_website zuweisen
1. Weiterleitung von testing.m-f.ch auf www.m-f.ch aufheben

Unter Websites & Domains: bei der Domain testing.m-f.ch, unter Hosting-Einstellungen, den Hosting-Typ auf Website-Hosting setzten. Der Domain Name bleibt so wie er vorausgefüllt ist (testing.m-f.ch). Der Dokumentenstamm bleibt auch beim default Wert.  

2. Die aktuelle Webseite auf testing.m-f.ch kopieren

Unter Websites & Domains: bei der Domain m-f.ch, auf Website kopieren klicken. Website in Plesk angeben, und der Name der Website sollte testing.m-f.ch sein. 

3. Die Datenbank mf_website in die Datenbank testing_mf_website kopieren

Unter Datenbanken: bei mf_website auf Kopieren klicken. Als Zieldatenbank die vorhandene Datenbank testing_mf_website wählen. 

4. testing.m-f.ch die Datenbank testing_mf_website zuweisen

Unter Websites & Domains: bei mf_testing unter Datenbanken "Vorhandene Datenbank auswählen" anklicken, und testing_mf_website auswählen. 

Bemerkungen

  • Eine detailliertere Beschreibung der Schritte findet man auf: https://docs.plesk.com/en-US/obsidian/customer-guide/websites-and-domains/extended-website-management/working-with-a-staging-site.66505/
  • Sobald man mit dem Testen fertig ist, sollte die Domain testing.m-f.ch wieder auf www.m-f.ch weitergeleitet werden.
