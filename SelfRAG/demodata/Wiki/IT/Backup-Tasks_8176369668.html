IT : Backup Tasks

Created by Ruedi Steinmann, last modified by nizami.sevindi on Aug. 26, 2024

Weekly

Überprüfen, ob die Heartbeat-Benachrichtigung im Teams angekommen ist:

  • Im Teams:

    • Teams

    • Cudos AG

    • Backup Monitoring (evtl. über “1 ausgeblendeter Kanal”

Monthly

  • Einloggen auf Backup PC (user/user)

  • sudo btrfs device stats /data

    alle Werte sollten 0 sein

  • sudo btrfs fi show /data

    Beide Disks sollten mit dem gleichen used Speicher angegeben sein

  • Jeweils am 25. des Monats startet ein cron job ein disk scrub. Der Status muss überprüft werden

    sudo btrfs scrub status /data

    Wichtig:

    • Scrub started: muss zu letztem Monat passen

    • Status: finished

    • Error summary: no errors found
