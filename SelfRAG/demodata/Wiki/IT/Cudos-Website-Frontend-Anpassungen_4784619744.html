Website Backup

Ein Backup der Cudos Website kann über das Akeeba Backup Plugin im Joomla Frontend vorgenommen werden.

Typischerweise wird vor und nach Änderungen auf der Webseite ein Backup erstellt und auf folgendem Netzwerkpfad abgelegt:

G:\Geteilte Ablagen\Diverses\Verkauf-Marketing\Website\Backup

Website Restore

Ein mit Akeeba Backup erstelltes Backup kann am einfachsten mit Akeeba Kickstart wiederhergestellt werden.

Vorgehen:

  1. Akeeba Kickstart downloaden.
  2. Akeeba Kickstart Files per FTP in Website Ordner legen.
  3. Gewünschtes Backuparchivfile (.jpa) ebenfalls in Website Ordner legen.
  4. Akeeba Kickstart Restore Prozess über Aufruf von http://www.m-f.ch/kickstart.php.
  5. Gemäss Aufforderungen Backup einspielen.
