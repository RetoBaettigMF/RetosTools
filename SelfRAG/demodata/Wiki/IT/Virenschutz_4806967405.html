IT : Virenschutz

Created by Reto Bättig, last modified on Aug. 02, 2018

Bis im Sommer 2018 haben wir Symantec Endpoint Security verwendet. 

Die Lizenzen wurden danach nicht mehr erneuert, da Microsofts hausinterne Lösung mittlerweile gleich gut ist (siehe z.B. https://www.heise.de/security/meldung/Win-Defender-Gratis-Virenschutz-zieht-im-Test-mit-Konkurrenzprodukten-gleich-4123805.html)

Wichtig: Alle Mitarbeitenden sind selbst dafür verantwortlich, dass der Windows Defender (bzw. Security Essentials) aktiviert und auf dem aktuellen Stand ist. Bitte Mitarbeitende unbedingt darauf hinweisen!

Windows 10

Ab sofort wird also standardmässig der Windows Defender eingesetzt. Alle Mitarbeitenden sind selbst dafür verantwortlich, dass die der Windows Defender (Virenschutz + Firewall) nicht ausgeschaltet ist und dass er aktuelle Virensignaturen installiert hat.

Das Windows Defender Security Center sollte in etwa so aussehen:

Windows 7

Auf älteren Rechnern mit Windows 7 müssen entsprechend die Microsoft Security Essentials installiert und à Jour gehalten werden: https://www.microsoft.com/de-ch/download/details.aspx?id=5201
