Random Whitespace loswerden

Auf verschiedenen Seiten gab es das Problem, dass in folgendem Code, willkürliche min-height definiert worden ist, und so random whitespace auf unserer Webseite zu sehen gewesen ist

<div id="tm-main" class="tm-main uk-section uk-section-default" uk-height-viewport="expand: true" style="min-height: 193.109px;"> ... </div>

Um dies loszuwerden, geht man unter Templates → Yootheme → index.php. Dann kann man Zeile 136

<div id="tm-main" <?= $view->attrs($attrs_main_section) ?> uk-height-viewport="expand: true">

durch

<div id="tm-main" <?= $view->attrs($attrs_main_section) ?>>

ersetzen.

Lösung stammt von: https://yootheme.com/support/question/136474 . Wir haben aber kein Child-Theme erstellt.
JOA: Ob dies wirklich das Problem gewesen ist, oder ob es nur ein Workaround ist, welches das Problem löst, ist nicht wirklich klar. Sollten sicher unsere Augen offen halten, falls wir mal etwas finden, was komisch ausschaut.

Video einbetten

Eingebettete Youtube Videos sind nicht per se responsive. Um die Videos in allen Formaten der Webseite schön darzustellen, haben wir ein einen custom video container im CSS definiert. Benutzerdefiniertes CSS kann man im Yootheme Pro unter Einstellungen → Benutzerdefinierter Code einfügen. 

Um ein Youtube Video einzufügen, kann man die folgende Anleitung befolgen: https://www.joomla-update.at/joomla-tipps-tricks/youtube-video-responsive-einbinden

  1. Auf Youtube das Video finden, und bei Teilen auf "Einbetten" klicken. Danach nicht vergessen, das Häckchen bei "Erweiterten Datenschutzmodus aktivieren" zu setzten. 

  2. iframe Snippet kopieren und die Höhen und Seitenangaben entfernen.

  3. iframe Snippet mit <div class="video-container"> und </div> umrahmen. 

  4. An die gewünschte Stelle einfügen. 

Code Snippets in Artikeln ergänzen

HTML Code von dem zu publizierenden Code erstellen, z. B. mit: https://pinetools.com/syntax-highlighter. 
Dann kann man den HTML Code in den Artikel ergänzen.

2 Spalten ohne Einrückung

Wenn man zwei Spalten machen möchte, sollte man davon absehen, "clearfix" in der Klasse zu gebrauchen. Mit <div class="features mainfeature2 clearfix"> werden die Spalten eingerückt. Wenn man nur <div class="features mainfeature2"> braucht, werden die default-Seitenränder des Moduls gebraucht.

Optimierte Bilder hochladen

Schritt 1: Bereiten Sie Ihre Bilder vor
Bevor Sie mit der Optimierung Ihrer Bilder beginnen, stellen Sie sicher, dass Sie die Bilder, die Sie für Ihre Joomla-Website verwenden möchten, auf Ihrem Computer bereit haben.


Schritt 2: TinyPNG
Gehen Sie auf die TinyPNG Website (https://tinypng.com/ ) , Sie können den kostenlosen Plan benutzen, der Ihnen erlaubt, eine begrenzte Anzahl von Bildern pro Monat zu komprimieren.


Schritt 3: Bilder mit TinyPNG komprimieren
Klicken Sie auf die Schaltfläche "Hochladen".

  • Wählen Sie die Bilddateien (JPEG oder PNG), die Sie optimieren möchten. Sie können auch mehrere Bilder auf einmal auswählen.

  • Warten Sie, bis TinyPNG die Bilder komprimiert hat. Dabei wird die Dateigröße automatisch reduziert, während die Bildqualität erhalten bleibt.

  • Laden Sie die optimierten Bilder auf Ihren Computer herunter.


Schritt 4: Convertio
Rufen Sie die Website von Convertio auf (https://convertio.co/ ). Wie TinyPNG bietet auch Convertio einen kostenlosen Plan mit Einschränkungen an.


Schritt 5: Bilder in das WebP-Format konvertieren
Klicken Sie auf die Schaltfläche "Dateien auswählen".

  • Wählen Sie die optimierten Bilder (entweder JPEG oder PNG), die Sie in das WebP-Format konvertieren möchten.

  • Wählen Sie "WebP" als Zielformat.

  • Klicken Sie auf die Schaltfläche "Konvertieren".

  • Warten Sie, bis Convertio die Bilder konvertiert hat. Sobald die Konvertierung abgeschlossen ist, erhalten Sie einen Download-Link.

  • Laden Sie die WebP-Bilder auf Ihren Computer herunter.


Schritt 6: Bilder zu Joomla hochladen
Jetzt, da Sie Ihre Bilder optimiert und in das WebP-Format umgewandelt haben, können Sie sie auf Ihre Joomla-Website hochladen
