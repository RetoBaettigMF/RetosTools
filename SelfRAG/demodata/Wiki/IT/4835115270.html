Arbeiten mit Google Drive

Google Drive hält normalerweise alle Daten in der Cloud und nicht auf der lokalen Festplatte. Dies ist nötig, weil wir insgesamt >1TB geteilte Daten haben und nicht alle diese Daten auf die lokale Festplatte synchronisieren wollen.

Solange man mit dem Internet verbunden ist und die Verbindungsgeschwindigkeit gut genug ist, lässt sich so gut arbeiten. Uploads werden zwischengespeichert und im Hintergrund auch über langsamerer Leitungen vernünftig synchronisiert.

Daten, auf die man häufig zugreift, können jedoch bewusst auf der lokalen Festplatte gehalten werden. 

Gleiches gilt für Daten, auf die man unterwegs - auch ohne Internetverbindung - zugreifen möchte.

Daten offline verfügbar machen

Daten können nur online oder auch offline verfügbar sein. Welche Daten offline verfügbar sind, sieht man am Status-Icon:

  • Nur Online verfügbar
  • Im Moment am Synchronisieren
  • Offline verfügbar (und damit auch schnell)

Um Daten offline verfügbar zu machen - was auch den Zugriff darauf deutlich schneller macht - geht man folgendermassen vor:

  • Rechtklick auf gewünschten Ordner (in diesem Fall "Vorlagen")
  • Auf "Drive File Stream" => "Offline verfügbar" klicken
  • Falls man später wieder lokal Speicherplatz freigeben möchte, kann man nicht mehr offline benötigte Ordner wieder auf "Nur online" setzen und der Platz wird automatisch freigegeben.


Alte Versionen wiederherstellen

In Google Drive werden automatisch alle alten Versionen bis zu einem alter von 30 Tagen gespeichert. Man kann jederzeit auf eine ältere Version zugreifen.

Der Zugriff auf alte Versionen funktioniert über das Web-Interface:

https:\\drive.google.com

Hier navigiert man zum gewünschten File (geht auch über die Volltextsuche).

Ein Rechtsklick auf die Datei öffnet das Kontextmenu und man kann "Versionen verwalten" anklicken:

 => 

Im erscheinenden Fenster "Versionen verwalten" sieht man alle verfügbaren älteren Versionen, wer sie bearbeitet hat und wann.

Über das Drei-Punkte Menu kann mann dann wählen, welche ältere Version man herunterladen möchte. Um die aktuelle Version durch eine ältere zu ersetzen kann einfach die ältere Version heruntergeladen werden und danach kann man dadurch die aktuelle Version überschreiben.

Wichtige Version für immer behalten

Ältere Versionen werden automatisch gelöscht, wenn sie älter als 30 Tage sind. Um eine bestimmte Version auch danach noch zu behalten (z.B. wenn man eine Version eines Dokumentes einem Kunden geschickt hat als Teil eines Vertrages), kann man die Version festhalten, indem man im oben genannten Menu bei der entsprechenden Version auf "Niemals löschen" klickt.

Datei finden

Die Online-Variante von Google Drive verfügt über eine sehr schnelle Volltextsuche. Dazu einfach im Suchfenster den gesuchten Begriff eingeben und sofort alle Dateien sehen:



Gelöschte Daten wiederherstellen

Gelöschte Daten werden für 30 Tage in den Papierkorb gelegt und können von dort wieder hergestellt werden.

Dies funktioniert jedoch nur über das Web-Interface: https://drive.google.com/drive/shared-drives

Grosse Datenmengen verschieben oder löschen

Da die Synchronisierung von Google File Stream nicht extrem schnell ist, funktionieren solche Aktionen sehr langsam, wenn man sie im Windows-Dateisystem macht (jede einzelne Datei muss angefasst werden über das Netzwerk). Viel schneller geht es, wenn man diese Funktionen über das Web-Interface: https://drive.google.com/drive/shared-drives durchführt.



Hinweis zu Spracheinstellungen

Unsere Anleitungen gehen alle davon aus, dass Deutsch als Systemsprache eingestellt ist. Dadurch ergeben sich auch sämtliche Verweise auf Dokumente in Anleitungen.

Wenn eine anderes Sprache (z.,B. Englisch) als Systemsprache eingestellt ist, funktionieren Links aus dem Wiki oder anderen Tools ev. nicht richtig, da die Hauptverzeichnisse anders heissen:

DeutschEnglisch
G:\Geteilte Ablagen\AdminG:\Shared drives\Admin
G:\Meine AblageG:\My Drive


Die Namen der Grundverzeichnisse ("Geteilte Ablagen"/"Meine Ablage") können leider momentan nicht eingestellt oder verändert werden, sondern sind von Google fix vorgegeben.
