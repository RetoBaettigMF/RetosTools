IT : Zweite MS Teams Instanz

Created by Simeï Steiner, last modified by nizami.sevindi on Aug. 26, 2024

Man kann mittels einem Script eine zweite Microsoft Teams Instanz öffnen. Somit kann man beispielsweise eine Instanz mit dem Account der Partnerfirma offen haben und eine Instanz mit dem Account der Cudos. Somit bleibt einem erspart Teams im Browser auszuführen.

Anleitung

  1. Versichere dich das Teams offen ist.

  2. Kopiere folgenden Code ins Notepad.

  3. Speichere das Text File z.B. als 'M_F_Teams.bat'.

  4. Starte das Script (per Doppelklick auf das .bat File).

  5. Logge dich im Teams mit dem gewünschten Account ein.

Zum öffnen der zweiten Instanz jeweils das Script starten, nachdem die erste Instanz offen ist.

Anmerkung: Du musst dich dann nicht jedesmal einloggen.

@ECHO OFF

REM Uses the file name as the profile name
SET MSTEAMS_PROFILE=%~n0
ECHO - Using profile "%MSTEAMS_PROFILE%"

SET "OLD_USERPROFILE=%USERPROFILE%"
SET "USERPROFILE=%LOCALAPPDATA%\Microsoft\Teams\CustomProfiles\%MSTEAMS_PROFILE%"

REM Ensure there is a downloads folder to avoid error described at
REM https://gist.github.com/DanielSmon/cc3fa072857f0272257a5fd451768c3a
mkdir "%LOCALAPPDATA%\Microsoft\Teams\CustomProfiles\%MSTEAMS_PROFILE%\Downloads"

REM Fixes missing "Desktop" folder
mkdir "%LOCALAPPDATA%\Microsoft\Teams\CustomProfiles\%MSTEAMS_PROFILE%\Desktop"

ECHO - Launching MS Teams with profile %MSTEAMS_PROFILE%
cd "%OLD_USERPROFILE%\AppData\Local\Microsoft\Teams"
"%OLD_USERPROFILE%\AppData\Local\Microsoft\Teams\Update.exe" --processStart "Teams.exe"
