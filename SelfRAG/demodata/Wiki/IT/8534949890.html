IT : Combox Empfang (inkl. Ferienansage)

Created by Reto Bättig, last modified on Dez. 22, 2023

Die Combox bzw. Mailbox für den Empfang wird auf der Telefonnummer 444 in 3cx eingerichtet:

Ruth kann die Ansage selber einstellen bei den 3cx-Einstellungen:

Achtung:

  • Die Empfangsnummer (444) muss auf abwesend gestellt werden

  • Die Anrufe gehen zuerst auf den Ringruf (Signalisierungsgruppe 901) und erst danach auf die Combox (gewollt, falls jemand doch im Büro ist)

  • In der Ringruf-Gruppe muss die Signalisierungsdauer heruntergestellt werden auf 10s. Darüber funktioniert es aus unerklärlichen Gründen nicht: (Das kann nur der Admin: https://mfengineering.3cx.ch:5001/)

Der Admin kann die Ansage ebenfalls wechseln, indem er auf die Nummer 444 geht:

Sie kann auch durch die Admin geändert werden.

Für Ferien/Weihnachten können unter “Mailbox-Ansagen” spezielle Ansagen ausgewählt werden.
