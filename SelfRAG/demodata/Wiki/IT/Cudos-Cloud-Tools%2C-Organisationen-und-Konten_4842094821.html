IT : Cudos Cloud Tools, Organisationen und Konten

Created by Reto Bättig, last modified by nizami.sevindi on Aug. 26, 2024

Diese Seite enthält eine Liste und Links mit Zugangsdaten für alle Tools und Konten, die im Zusammenhang mit der Arbeit bei Cudos verwendet werden und nicht von unseren Kunden selbst verwaltet werden. Oft können in einem Tool Organisationen eingerichtet werden, welche die Assets einer Firma verwalten.

Prinzipien:

  • Von jedem Tool muss mindestens eine zuständige Person ersichtlich sein

  • Entweder wird das Admin-Login inkl. Passwort beschrieben (ev. lesegeschützt für IT-Mitarbeiter)...

  • ...oder mindestens 2 Cudos Mitarbeiter müssen Admin zugang haben, wobei alle MA mit Admin Zugang namentlich erwähnt werden müssen

  • Bei der Abonnierung von neuen Tools ist die IT zu informieren und das Tool ist hier aufzuführen
