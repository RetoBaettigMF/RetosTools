IT : Google Sheets Automatisierung mit AI und RolX

Created by Reto Bättig, last modified on Sept. 26, 2024

Für Google Sheets wurden Bibliotheken entwickelt, um Automatisierungen von Abfragen, Auswertungen etc. zu vereinfachen. Die Bibliotheken können im Apps Script eingebunden werden.

Entwickelt und gewartet werden die Bibliotheken momentan von Reto Bättig

Cudos AI Library

Google Link: https://script.google.com/home/projects/11ctRQZSAFBVQpk9lynQOR9Ye1BjvtbvL3MfE_4LWE3y5sHVQCKI92Vih/edit

Script ID: 11ctRQZSAFBVQpk9lynQOR9Ye1BjvtbvL3MfE_4LWE3y5sHVQCKI92Vih

Funktionen:

  • getCompletionAzure(args, deployment):

    • Sendet eine Anfrage an die Azure OpenAI API, um eine Antwort zu erhalten.

    • Optional kann ein Deployment angegeben werden. Wenn nicht, wird ein Standard-Deployment verwendet.

    • Rückgabe: API-Antwort als JSON-Objekt oder Fehlermeldung.

  • getSingleCompletionAzure(prompt):

    • Sendet einen einzelnen Prompt an die Azure OpenAI API und gibt die Antwort als Text zurück.

    • Protokolliert den Aufruf mit der LogCall-Funktion.

  • SheetGPTRange(prompt, range):

    • Sendet eine Anfrage an die API und erwartet eine Antwort basierend auf einem Prompt und optionalen Daten in einem Bereich.

    • Versucht, die Antwort als JSON-Array zu parsen.

    • Rückgabe: Geparstes JSON-Array oder rohe Antwort.

  • SheetGPT(prompt, range):

    • Sendet eine Anfrage an die API mit einem Prompt und optionalen Daten in einem Bereich.

    • Gibt die rohe API-Antwort zurück, ohne Parsing in ein JSON-Array.

Cudos Controlling Library

Google Link: https://script.google.com/home/projects/1zpZczeKyxaEmCRXqplbV0HTUj8rM8itfV0Jn9WXJr_4_5iUAcGQade9N/edit

Script ID: 1zpZczeKyxaEmCRXqplbV0HTUj8rM8itfV0Jn9WXJr_4_5iUAcGQade9N

Funktionsbeispiel:

function GetRolXData() {
  var daterange = CudosControllingLibrary.getDateRangeLastMonth()
  var data = CudosControllingLibrary.getRolXData(daterange)
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Zeiterfassung");
  CudosControllingLibrary.clearSheetAndFillData(data, sheet)
}
