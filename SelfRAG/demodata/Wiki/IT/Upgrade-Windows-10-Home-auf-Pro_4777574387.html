IT : Upgrade Windows 10 Home auf Pro

Created by Ehemaliger Benutzer on März 09, 2016

Problem

ein (vorinstalliertes) Windows 10 Home kann nicht direkt mit einem Windows 7 Pro Schlüssel upgraded und aktiviert werden.

Lösung

Siehe Artikel How to upgrade from Windows 10 Home to Pro without hassles

Dabei wird der Default Windows Key VK7JG-NPHTM-C97JM-9MPGT-3V66T verwendet um das Upgrade zu initiieren.
