Cudos Trail : CyOne: Interview

Created by Rachel Blaser on Juni 22, 2020

Interview-Fragen

Anmerkungen

Dauer technischer Teil des Interviews: ca. 2 Stunden

Die unterstehenden Fragen zu den 6 Themenbereich sind nicht vollständig und nur aus dem Gedächtnis rekonstruiert.

gemäss Feedback liegt der Fokus stärker auf OO-Themen/Datenstrukturen/Patterns und Design/Testing wie auf WPF/Datenbanken

Themen/Fragen

Objekt-Orientierung allgemein

    • Was ist eine Klasse?
    • Was ist Vererbung?
    • Was ist Polymorphie?
    • Was bedeutet private/protected/public?
    • Beispiel mit 3 Klassen ausgedruckt (C erbt von B erbt von A), überladene Konstruktoren, Frage nach Aufrufreihenfolge / was passiert bei Aufruf einer Methode
    • zeichne UML Diagramm für obigen Fall

Testing

    • Fragen zur Test-Pyramide / Test-Arten
    • Was ist ein Unit Test? / Was macht einen guten Unit-Test aus?
    • Beispiel mit ausgedruckter Klasse (C#) und Tests dazu - Fragen nach Qualität der Tests / Code Coverage / Verbesserungsmöglichkeiten

Datenstrukturen

    • Erkläre die Datenstrukturen Array/List/Linked List/Hashtabelle/Baum/Graph
    • Implementiere eine Linked-List (Syntax nicht so relevant, z.B. C#) am Whiteboard

C# / WPF

    • Frage nach Patterns von WPF
    • (ausgedruckt Code einer kleinen WPF Applikation, erkläre die Elemente/zeige Verbesserungen auf)
    • Was sind Bindings in WPF / erkläre Binding Arten?
    • Was sind Resourcen in WPF, wie werden sie gut eingesetzt?

Patterns und Design

    • Was für Patterns kennst du?
    • Implementiere Singleton-Pattern am Whiteboard (Syntax nicht so relevant, z.B. C#)
    • Was ist MVVC? Was ist MVC? Observer Pattern?

Datenbanken

    • Was ist eine relationale Datenbank? Was für Alternativen gibt es?
    • Erkläre Vorteile einer relationalen DB?
    • Was für Datenbank-Typen kennst du?
    • Beispiel mit kleiner Datenbankstruktur - schreibe Query für gewisse Frage in T-SQL
