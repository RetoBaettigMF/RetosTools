Verknüpfung mit Google Drive Dateien und Ordner

Achtung: Verknüpfungen mit Google-Drive Inhalten müssen richtig gemacht werden, damit nicht aus Verstehen Vorlagen etc. überschrieben werden im Browser.


Vorgehen:

  • Ordner im Browser in Google Drive öffnen (https://drive.google.com/drive/shared-drives)
  • Gewünschte Datei rechtsklicken und auf "Get link" clicken:

  • Der Link sieht nun z.B. so aus: "https://docs.google.com/spreadsheets/d/1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9/edit?usp=sharing&ouid=105634099053787135033&rtpof=true&sd=true"

  • Vom Link ist nur der fett markierte Teil zwischen /d/.... und dem Fragezeichen interessant (in diesem Fall "1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9"

  • Diesen Teil ausschneiden und bei den Punkten einfügen im folgenden Link: "https://drive.google.com/file/d/..." => "https://drive.google.com/file/d/1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9"

Alternativ kann auch mit Google Docs weitergearbeitet werden (Beschreibung der verschiedenen URL-Varianten für einen Google Drive Link):

  • Jetzt den Link bearbeiten (z.B. in einem Text-Editor und alles ab "edit?" löschen => sieht nun so aus: "https://docs.google.com/spreadsheets/d/1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9/"
  • Jetzt noch die gewünschte Funktion an den Link anhängen und diesen dann in's Wiki kopieren:
    • /preview => Datei wird im Browser angezeigt, kann aber nicht bearbeitet werden => Link: "https://docs.google.com/spreadsheets/d/1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9/preview"
    • /copy => Datei wird kopiert und kann dann vom Benutzer bearbeitet werden => Link: "https://docs.google.com/spreadsheets/d/1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9/copy"
    • /template/preview => Datei wird als Template verwendet und Benutzer kann daraus neues File erstellen  => Link: "https://docs.google.com/spreadsheets/d/1_33GmOUUMWAnoYJp4FXMwSi_vXZSpfe9/template/preview"
    • /edit => ACHTUNG: Das überschreibt dann das Original-File und das will man in 99% der Fälle nicht (ausser z.B. für den Ressourcenplan)
